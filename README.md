# Job Listings Web Scraper
A Python-based web scraper that extracts job listings from websites and stores them in structured formats for analysis.

## Features
- Scrapes job title, company name, and location
- Handles multiple pages (pagination)
- Stores data in CSV and JSON formats
- Error handling for missing fields

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Pandas

## Output
The scraper generates:
- jobs.csv
- jobs.json

## How to Run
1. Clone the repository
git clone https://github.com/mahishika/job-listings-web-scraper.git

2. Install dependencies
pip install -r requirements.txt

3. Run the scraper
python scraper.py
